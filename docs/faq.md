# FAQ

## What is the scope of this wiki?

This project documents verified game metadata: release dates, platforms, genres, developers, publishers, and official references. It is not a strategy guide or review site.

## How do I add a new game entry?

Use the template in `templates/game-entry.md`, fill in verified fields, and open a pull request with citations.

## Can I copy content from Wikipedia?

You may use Wikipedia as a reference, but do not copy content verbatim without attribution. Always link to the source.
