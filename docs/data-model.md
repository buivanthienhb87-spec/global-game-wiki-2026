# Data Model

The wiki uses simple records so contributors can add verified game information without a complex database. Records should remain source-backed and easy to translate.

| Field | Meaning |
|---|---|
| `title` | Display name of the game. |
| `slug` | Stable identifier for filenames and URLs. |
| `platforms` | Supported platforms from verified sources. |
| `genres` | Broad genre labels. |
| `references` | URLs used to verify the entry. |
