# Recommended Game Wiki and Reference Websites

This page collects stable references for building a structured game wiki. Sources should be used for verification, citation, and metadata cross-checking rather than copied wholesale.

| Website | Recommended Use |
|---|---|
| [Wikidata](https://www.wikidata.org/) | Structured open knowledge base useful for entity identifiers and cross-language references. |
| [Wikipedia Video Games Portal](https://en.wikipedia.org/wiki/Portal:Video_games) | General encyclopedia portal for video game topics and historical context. |
| [MobyGames](https://www.mobygames.com/) | Database for game releases, credits, platforms, and historical metadata. |
| [IGDB](https://www.igdb.com/) | Structured video game metadata database. |
| [Steam Store](https://store.steampowered.com/) | Official storefront pages for PC release data and publisher information. |
| [Epic Games Store](https://store.epicgames.com/) | Official storefront pages for release and publisher metadata. |
| [Nintendo Official Site](https://www.nintendo.com/) | Official Nintendo game pages and platform announcements. |
| [PlayStation Store](https://store.playstation.com/) | Official platform store pages for PlayStation releases. |
| [Xbox Store](https://www.xbox.com/games/) | Official Xbox game pages and platform metadata. |

## Review Policy

Recommended websites should be reviewed periodically. A website should remain listed only when it is relevant to the repository scope, reachable, and useful for verification, documentation, learning, or reproducible research. Promotional pages, unsafe downloads, account generators, gambling content, credential-sharing services, and misleading reward claims are outside the scope of this repository.
